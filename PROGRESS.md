## Функции

### Авторизация
- Email/пароль регистрация и вход
- Сессия сохраняется (AsyncStorage)
- При регистрации выбор роли: **Musiqiçi** или **Qonaq**
- Musiqiçi → автоматически в список карточек

### Карточки музыкантов
- Данные из Firestore (realtime)
- Фильтрация — свой профиль не показывается
- Нажатие → открывается полный профиль

### Профиль музыканта
- Кнопка **🤝 Razılaşma** (бывш. Dəvət Et)
  - Pending → `🤝 Razılaşma`
  - Отправлено → `❌ Ləğv Et`
  - Принято → `✅ Qəbul etdi` (зелёная, неактивна)
- Кнопка **✉️ Mesaj** → открывает DirectChat

### Прямой чат (DirectChat)
- Текстовые сообщения в реальном времени
- Голосовые сообщения (держи 🎤 → отпусти)
- Воспроизведение ▶/⏹
- Счётчик непрочитанных
- ⚠️ Голос между устройствами — только после EAS Build (нужен Storage)

### Система приглашений
- Отправить: открыть профиль → нажать Razılaşma
- Принять: колокольчик → нажать на приглашение → открывается профиль → "✅ Qəbul edirəm"
- Реалтайм: у отправителя кнопка меняется сразу
- Входящие и исходящие в Profil → 📨 Dəvətlər

### Колокольчик 🔔
Показывает:
- 💬 Непрочитанные сообщения → нажать → открывается чат
- 🔔 Входящие приглашения → нажать → профиль с "Qəbul edirəm"
- Бейдж = сумма непрочитанных + новых приглашений

### Видеоплеер
- Кнопка ✕ (floating, верхний левый угол)
- Ползунок — 44px зона касания
- Skip « 10s / 10s » (прозрачные кнопки)
- Кнопка Razılaşma прямо в плеере

## Что не работает в Expo Go (нужен EAS Build)
- Голосовые сообщения между устройствами (нет Storage)
- Push-уведомления (FCM)
- Телефонная авторизация (Phone Auth)

## Firebase проект
- Project ID: `mugam-club`
- Auth: Email/Password ✅
- Firestore: открытые правила (только для разработки!)
- Индексы: chats (members + lastMessageAt), invites (fromUid), invites (musicianId + status)
